# ✦ NECTAR — Premium Functional Wellness Tonic
**A High-Fidelity Web Experience built with the Codvyn Workflow.**

NECTAR is a "Soft Brutalist" landing page designed to push the boundaries of modern UI builders. It leverages physics-based motion and 3D depth to create a sanctuary-like digital experience.

## 🚀 Key Features
* **3D Bottle Engine:** Physics-based rotation and Y-axis scaling using GSAP.
* **Physics Scrolling:** Immersive momentum scrolling powered by Lenis.js.
* **PULSE Interactions:** Custom horizontal-scroll ingredient sections.
* **Cinematic Visuals:** SVG grain overlays and glassmorphic depth layering.

## 🛠️ Tech Stack
* **Framework:** Next.js 14 (App Router)
* **Animation:** GSAP (GreenSock) & ScrollTrigger
* **Smooth Scroll:** Lenis.js
* **Styles:** CSS Modules (Manual Creative Engineering)

## 📦 Installation & Setup
1. Clone the repo: `git clone https://github.com/shashanthnetha/nectar-tonic.git`
2. Install dependencies: `npm install`
3. Run local server: `npm run dev`
